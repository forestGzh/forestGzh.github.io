Razor

  • Gzh
  • 2 Minutes
  • 2018年3月19日

Razor是asp.net MVC 3.0开始出现的，是服务器端标记语言，是在网页中嵌入服务器代码（C#等）。

Razor语法

  • Razor代码写在@{ }中。
  • 行内表达式以@开头。
  • 语句以分号结尾。
  • 字符串由引号包围。
  • C#代码对大小写敏感。
  • C#文件后缀名是 .cshtml

Razor布局页面

整体视图模板，_layout.cshtml，@RenderBody( )是模板页的“占位符”，子页所有html代码都将替换到模板页的@RenderBody( )处。
如果要多个“占位符”，可以用@RenderSection(“”)，子页可以用@section { }
View目录下自动添加的_ViewStart.cshtml文件代码优先于任何视图，所以任何一个视图都可以重写LayOut属性来指定自己想要的模板布局页面。

ViewStart这个文件代码优先于其他视图，任何视图都可以重写Layout属性来制定自己想要的布局页面。

如果视图不指定Views/Shared下的布局，则使用ViewStart下的布局，如果指定了Views/Shared下的布局，就使用Views/Shared的布局。

Action方法可以通过 PartialView方法以 PartialViewResult形式返回分部视图,分部视图的作用一般用于嵌到一些正常的视图中去。
Ajax请求中jQuery的load()方法可以获取分部视图。

如果分部视图只是一些简单的html代码，使用RenderPartial，如果除了有html代码外，还要通过读取数据库的数据来渲染，就必须用RenderAction，因为它可以在Action里调用Model里的方法读取数据库，渲染视图后呈现。
