Java类集

  • Gzh
  • 2 Minutes
  • 2017年8月12日

使用对象数组保存一组对象有许多限制，类集其实就是一个动态的对象数组，对一些实现好的数据结构进行了封装，使用很方便，而且类集框架不受对象数组长度的限制。

整个Java类集中最常使用的类集接口有：Collection、List、Set、Map、Iterator、ListIterator、Enumeration、SortedSet、SortedMap、Queue、Map.Entry

Collection

存放一组对象的最大接口，一般不直接使用。

List

Collection接口的子接口，是最常用的接口，对Collection接口有很多的扩充，里面的内容允许重复。

Set

Collection接口的子类，没有扩充，不允许内容重复。

Map

存放一组键值对的最大接口，以key-value形式保存。

Iterator

集合的输出接口，只能单向输出。

ListIterator

Iterator的子接口，可以双向输出。

Enumeration

最早的输出接口，用于输出指定集合中的内容。

SortedSet

单值的排序接口，实现此接口的集合类，里面的内容可以使用比较器排序。

SortedMap

一对值的排序接口，实现此接口的集合类，里面的内容可以使用比较器排序。

Queue

队列接口，此接口的子类可以实现队列操作。

Map.Entry

Map.Entry的内部接口，每个Map.Entry对象都保存着一对key-value的内容，每个Map接口中都保存有多个Map.Entry接口实例。
